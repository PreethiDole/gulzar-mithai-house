# Gulzar Mithai House

A warm and elegant frontend website built for a fictional traditional sweet shop as part of a Frontend Developer technical assignment.

## 🌟 Brand Concept

Gulzar Mithai House is a Hyderabad-based family-owned sweet shop founded in 1995.  
The brand focuses on warmth, heritage, and handcrafted Indian sweets made with purity and tradition.

The website is designed to create appetite, warmth, and trust — in that order.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Navbar toggle & scroll animations)
- Google Fonts (Playfair Display & Poppins)

---

## ✨ Features

- Fully responsive design (Desktop / Tablet / Mobile)
- Sticky navigation bar
- Mobile hamburger menu
- Smooth scrolling
- Scroll reveal animations
- Clean typography-based menu layout
- Custom styled scrollbar
- Multi-page navigation (Home, Menu, About, Contact)

---

## 📱 Responsiveness

- Desktop (1280px+): Multi-column layout
- Tablet (768px–1279px): Adaptive layout
- Mobile (<768px): Single-column layout with accessible navigation

---

## 🎨 Design Decisions

- Primary Color: Deep Maroon (#8B0000)
- Accent Color: Gold (#D4AF37)
- Background: Warm Cream (#FFF8E7)
- Typography chosen to reflect traditional yet elegant branding.

The design focuses on visual warmth and readability rather than heavy imagery.
